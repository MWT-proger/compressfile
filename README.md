# CompresFile 

### Будущий микросервис
Позволит указав в URl 
1. путь к файлу 
2. параметры сжатия такие как ширину и высату

Сохранить в хранилище файл в указанном размере и вернуть его

В настоящий момент это демо версия собрана из кусков кода
